Math 9 — Linear Relations

Teacher Notes Package

Teaching reference — not a script

How to use this document

Each teaching beat has three parts:

Label What it contains
SHOW What is visible to students — write on the board or project. One thing per beat.
SAY Key phrases and questions. Not sentences to read — prompts for fluent delivery. Answers to blanks shown in [brackets].
CUE The exact signal for students to write. Four types: ★ Write / ✎ Draw / ⊡ Think / ↩︎ Check

Flags appear in callout blocks: > FLAG — Misconception: Red alert. Students will get this wrong without a direct address. > FLAG — Pacing: Warning for where lessons tend to lose momentum. > FLAG — Rule of Four: Reminder to name which representation you are currently working in.

The screen and the voice carry different things. What is SHOW stays visual — labels, equations, graphs, key terms. What is SAY carries the reasoning. Never put your SAY on the screen.

Pre-Knowledge Check

~10 minutes — before any instruction

Beat 1 — Opening SHOW: Board: “Write everything you think you know about linear relations and graphs.” (That’s it — no further prompts on the board.) SAY: - “Before we start the unit, I want to know what’s already in your head.” - “This is not a test. There are no marks. I am not going to correct it today.” - “Write honestly. You will come back to this page at the end of the unit.” - “Questions to think about: what is a linear relation? What makes something linear? What does a linear graph look like?” CUE: ★ Write: “Open your notes to page 1. You have 5 minutes — write everything you know. Go.”

FLAG — PacingDo not comment on student responses or read them aloud. The pre-knowledge check loses its value if students calibrate to what you reward. Circulate and look for: (a) students who already know y = mx + b, (b) students who think ‘linear’ just means ‘line’ without understanding constant rate of change, (c) students who have confused slope with y-intercept already. These patterns tell you where to spend time in Sections 2 and 3.

Beat 2 — Phone plan prompt SHOW: Board: “A phone plan costs $15/month plus $0.10 per text. How would you write a rule for the total cost?” SAY: - “Try to write this as an equation — or describe the pattern.” - “No correct answer required. I want to see what your instinct is.” CUE: ★ Write: “Write your rule for the phone plan. Try to use symbols if you can.”

Allow 3 minutes. Do not give any hints. The purpose is to surface pre-existing algebraic language — you will return to this exact context in Section 4.

Section 1 — Patterns and the Four Representations

Learning target: I can represent a linear pattern using a table of values, an equation, a graph, and a verbal description, and translate between all four forms. Timing: ~25 minutes direct instruction

Beat 1 — Vocabulary setup SHOW: Board: The six vocabulary terms in a column — Linear relation / Table of values / Independent variable / Dependent variable / Equation (of a relation) / Rule of Four SAY: - “Before we go into the notes — these six terms are on your vocab table.” - “Don’t fill them in yet. We’ll build them as we go through the section.” - “By the end you will have your own definition and an example you thought of yourself.” CUE: ⊡ Think: “Read the six terms. Which one do you think you already know? Keep it in your head.”

Beat 2 — What is a linear relation? SHOW: Board: LINEAR RELATION: equal changes in x produce equal changes in y. Draw a simple two-column table: x = 0, 1, 2, 3 and y = 2, 5, 8, 11. SAY: - “A linear relation is a relationship between two variables.” - “What’s the pattern here? What happens to y each time x goes up by 1?” - Pause 4–5 seconds. Get responses. - “Right — y goes up by 3 every time. Same change. Always. That’s what linear means.” - “Equal changes in x produce [equal] changes in y.” - “Not sometimes equal. Not approximately equal. Exactly equal, every time.” CUE: ★ Write: “A linear relation is a relationship between two variables where equal changes in one variable produce [equal] changes in the other.”

FLAG — Rule of FourYou are introducing the tabular representation here. Say explicitly: “We’re starting with the table — one of the four ways to show this relationship. The table shows us specific values and the pattern in the numbers.”

Beat 3 — The Rule of Four SHOW: Board: The four-row table — Verbal / Table / Equation / Graph. Leave the ‘What it shows’ column visible. SAY: - “There are four ways to represent the same relationship. We call this the Rule of Four.” - “Verbal — a sentence. Table — pairs of x and y values. Equation — an algebraic rule. Graph — a visual picture.” - “They all describe the SAME relationship. One is not better than another.” - “Your job in this unit is to be able to move between all four — fluently.” - “When you see a table, you should be able to sketch the graph. When you see an equation, you should be able to write the verbal description.” CUE: ★ Write: “Complete your Rule of Four table — read each row while I point to it.”

Beat 4 — Worked example: build all four representations SHOW: Board: Verbal: “A tree grows 4 cm per year. It was 10 cm tall when planted.” SAY: - “Let’s build all four representations from this single verbal description.” - “We already have the verbal. That’s representation one.”

Table: SHOW: Board: Draw the table — x (Year), y (Height in cm). Fill in x = 0 through 4. Leave y column blank. SAY: - “Year 0 — just planted. Height is 10 cm.” - “Year 1 — grows 4 cm. What’s the height?” - Pause. Students call out or write. “14. Year 2?” - “18. Year 3?” — “22. Year 4?” — “26.” - Fill in as you go: [14, 18, 22, 26]. CUE: ★ Write: “Fill in the height column: 10, [14], [18], [22], [26].”

FLAG — Rule of FourAnnounce: “That’s representation two — the table. Watch for the constant difference: plus 4 every time. That constant difference is what we will call the slope in Section 2.”

Equation: SHOW: Board: y = x + . Point to the blanks. SAY: - “Each year adds 4 cm — that goes in front of x.” - “We started at 10 — that’s the value when x is zero.” - “So the equation is y = 4x + 10.” - Write: y = 4x + 10. CUE: ★ Write: “y = [4] x + [10]”

FLAG — Rule of FourAnnounce: “Representation three — the equation. Two things to name: the 4 is the rate of change; the 10 is the starting value. We will spend all of Section 3 on what these mean in y = mx + b.”

Graph: SHOW: Board: Point to the coordinate grid on the student page. SAY: - “Plot the five points from your table. Then draw the line through them.” - “Notice: this is the same page as your table. The graph is showing the same information — just visually.” - “What shape does the graph make? A straight line. That’s the visual signature of a linear relation.” CUE: ✎ Draw: “Plot your five points and draw the line. Label the axes: x = Year, y = Height (cm).”

FLAG — Rule of FourAll four representations are now complete. SAY: “We started with a sentence and ended with a graph — same relationship, four different forms. Before we move on: what stays the same across all four? [The rate of change — +4 per step — appears in every form.]”

Self-explanation beat: SAY: “Before moving on — look at how we wrote the equation. In your own words: what did the starting value 10 do in the equation, and why did we put it there?” CUE: ★ Write: “Answer the ‘Explain’ prompt in your notes before looking ahead.”

Allow 60–90 seconds. You want: the starting value (10) becomes the constant added to the rate — it tells us where the relationship begins. Students who write “because that’s the answer” need a redirect: “Tell me what problem that number solves.”

Beat 5 — Independent and dependent variables SHOW: Board: Independent variable = what you CONTROL (input, x). Dependent variable = what DEPENDS on it (output, y). SAY: - “Two terms you’ll use for every relation.” - “Independent: what you choose or control. In the tree problem — you pick the year.” - “Dependent: what results from your choice. The height depends on the year.” - “Not hard to remember: ‘depend’ — the dependent variable depends.” CUE: ★ Write: “Independent variable: what you [control] (input, often x). Dependent variable: [depends] on it (output, often y).”

SHOW: Board: The three-row situations table — Cost/items, Distance/time, Temperature/time of day. SAY: - “In the cost situation — what do you control? The number of items. That’s independent.” - “What depends on it? The cost. Cost depends on how many items you buy.” - Work through all three rows together. CUE: ★ Write: “Complete the independent/dependent table — three situations.”

Answers: (1) independent = number of items, dependent = cost; (2) independent = travel time, dependent = distance; (3) independent = time of day, dependent = temperature.

Beat 6 — Translation task: equation to table SHOW: Board: y = −2x + 5. Draw the blank table beside it. SAY: - “Now we go the other direction — from equation to table.” - “Substitute x = 0 into the equation. What is y?” - Pause. [y = 5]. - “x = 1?” [y = 3]. “x = 2?” [y = 1]. “x = 3?” [y = −1].” - “What pattern do you see in the y-values?” - Pause 4–5 seconds. Get responses. - “They decrease by 2 each time. The negative in front of x tells us the y-values go down as x goes up.” CUE: ★ Write: “Complete the y = −2x + 5 table. Write the pattern you notice: [y decreases by 2 each time x increases by 1].”

FLAG — Rule of FourLabel this explicitly: “Equation → Table. We moved between two representations. The pattern in the table (−2 each step) matches the coefficient −2 in the equation. These are not separate facts — they’re the same thing.”

Section 1 — Retrieval Check Facilitation

Before releasing: SAY: “Close your notes. Everything closed. Retrieval check — write answers from memory.” CUE: ↩︎ Check: “Notes closed. Retrieval check begins now.”

After students finish (4–5 min): SAY: “Open your notes. Find the answers. Fix anything wrong or incomplete — use a different colour if you have one.” CUE: ★ Write: “Open your notes. Check and correct now.”

Answer guide: 1. Equation: C = 3.50h (where h = hours). Table: (0, 0), (1, 3.50), (2, 7.00), (3, 10.50), (4, 14.00). Graph: straight line starting at origin, rising to the right. 2. The equation is y = 3x + 5. (Slope: +3 each step. Starting value at x=0: y=5.) 3. The independent variable is the amount of fuel used. (Fuel is what you control — kilometres driven depends on how much fuel is consumed.) 4. In a linear relation, the y-values change by the same amount (constant difference) every time x increases by 1. 5. Stretch — Not linear. Differences: 5−3=2, 9−5=4, 15−9=6. The differences are not constant (they increase), so the table is NOT linear.

Q5 is the stretch — if students finish early, prompt: “Don’t check whether y changes — check whether y changes by the SAME amount each time.” The key insight is that constant first differences = linear; changing first differences = not linear.

Section 2 — Slope as Rate of Change

Learning target: I can calculate slope using the rise/run formula, interpret slope as a rate of change, and distinguish between positive, negative, zero, and undefined slopes. Timing: ~25 minutes direct instruction

Beat 1 — Vocabulary setup SHOW: Board: The eight vocabulary terms — Slope / Rise / Run / Rate of change / Positive slope / Negative slope / Zero slope / Undefined slope. SAY: - “Eight terms today. All connected to one central idea.” - “Don’t fill them in yet. We build them as we go.” CUE: ⊡ Think: “Read the list. Which one of these do you already know something about? Keep it in your head.”

Beat 2 — What slope measures SHOW: Board: SLOPE = how STEEP a line is AND which DIRECTION it goes. Draw two lines on the board — one steep, one shallow. SAY: - “Slope is a number that tells us two things about a line: how steep it is, and whether it goes up or down.” - “The steeper the line, the larger the absolute value of the slope.” - “The direction tells us whether it rises (positive) or falls (negative) as we move right.” CUE: ★ Write: “Slope measures how [steep] a line is and in what [direction].”

Beat 3 — Rise over run formula SHOW: Board: Draw a line segment with a clear staircase shape. Label the vertical leg: rise (change in y). Label the horizontal leg: run (change in x). Write: m = rise / run = (y₂ − y₁) / (x₂ − x₁). SAY: - “Rise: how far you go up or down. Measured on the y-axis.” - “Run: how far you go across. Measured on the x-axis.” - “Slope = rise divided by run. That’s the formula.” - “The formal version using coordinates: m equals y₂ minus y₁, divided by x₂ minus x₁.” - “It doesn’t matter which point is point 1 and which is point 2 — as long as you are consistent.” CUE: ★ Write: “slope = rise/run = change in y / change in x = (y₂ − y₁) / (x₂ − x₁)”

Beat 4 — Worked Example 1: from a graph (positive slope) SHOW: Board: Plot (1, 2) and (4, 8). Draw the line. Draw the staircase — label rise and run visually. SAY: - “Two points: (1, 2) and (4, 8). Let me label them: point 1 is (1, 2), point 2 is (4, 8).” - “Rise: y₂ minus y₁. That’s 8 minus 2. What is that?” - Pause. [6]. - “Run: x₂ minus x₁. That’s 4 minus 1. What is that?” - Pause. [3]. - “Slope = 6 divided by 3 = 2.” CUE: ★ Write: “Rise = [6] / Run = [3] / Slope = [6] / [3] = [2]”

SAY: “The slope of 2 means: for every 1 unit you move right, the line goes [2] units up. Can you see that staircase on the graph?” CUE: ★ Write: “The slope of 2 means: for every 1 unit right, the line goes [2] units up.”

Self-explanation beat: SAY: “Before moving on — look at Step 2, where we calculated the run. In your own words: what did we do there and why did we subtract instead of add?” CUE: ★ Write: “Answer the explain prompt now — what did Step 2 do and why.”

Allow 60–90 seconds. You want: run = change in x = how far apart the two x-coordinates are. We subtract to find the difference between them. Adding would give the sum, not the distance between them.

Beat 5 — Worked Example 2: negative slope SHOW: Board: Plot (0, 6) and (3, 0). Draw the line — it falls to the right. SAY: - “Two points: (0, 6) and (3, 0).” - “Slope = (y₂ − y₁) / (x₂ − x₁) = (0 − 6) / (3 − 0).” - “Numerator: 0 minus 6 = what?” - Pause. [−6]. - “Denominator: 3 minus 0 = 3.” - “Slope = [−6] / [3] = [−2].” CUE: ★ Write: “m = [−6] / [3] = [−2]”

SAY: “A negative slope means the line is going [down to the right] — it falls as you move left to right. What does −2 mean in the temperature context? For each hour that passes, temperature drops [2] degrees.” CUE: ★ Write: “A negative slope means the line is going [down to the right]. In context: a change of −2 per unit.”

Self-explanation beat: SAY: “Before moving on — look at the point (0, 6). In your own words: what is special about a point where x equals zero, and what will that tell us in Section 3?” CUE: ★ Write: “Answer the explain prompt — what is special about x = 0?”

You want: when x = 0, the point sits on the y-axis. This is the y-intercept — the starting value. Planting the seed for Section 3.

Beat 6 — Misconception: different points give different slopes SHOW: Board: Write three collinear points: (0, 1), (2, 5), (4, 9). Leave space for two slope calculations side by side. SAY: - “Here’s a misconception I want to address directly.” - “‘If I use different points on the line, will I get a different slope?’ Let’s check.” - “Using (0,1) and (2,5): rise = 5−1 = 4, run = 2−0 = 2, slope = 2.” - “Using (2,5) and (4,9): rise = 9−5 = 4, run = 4−2 = 2, slope = 2.” - “Same slope. For a straight line, slope is CONSTANT — every pair of points gives the same ratio.” - “In fact, this is what makes it a LINE. If the slope changed depending on which points you picked, the relationship would NOT be linear.” CUE: ★ Write: “Using (0,1)+(2,5): slope = [2]. Using (2,5)+(4,9): slope = [2]. Same? [Yes].”

FLAG — Misconception“If I use different points, I get a different slope.” Address directly with the worked example above. The key conceptual link: constant slope is the defining property of linearity. Students who pick non-collinear points from a table (or misread a graph) and get different slopes are diagnosing that the relation is NOT linear — that’s a useful error, not a mistake.

Beat 7 — The four slope types SHOW: Board: The four-column table — Positive / Negative / Zero / Undefined. Draw a sketch of each line type beside the label. SAY: - “Positive slope: line goes up as you move right. m > 0.” - “Negative slope: line goes down as you move right. m < 0.” - “Zero slope: line is perfectly horizontal — no rise. m = 0.” - “Undefined slope: line is perfectly vertical — no run. You’d divide by zero. m = undefined.” - Work through the context matching together: temperature flat all day → zero; a wall → undefined; ski hill dropping → negative. CUE: ★ Write: “Complete the four slope types table. Match: temperature = [zero] / wall = [undefined] / ski hill = [−3/10 or negative].”

FLAG — Misconception“A steeper line always has a larger slope value.” This is only true when both slopes are positive. Draw two lines on the board — one with slope +1 (gentle rise) and one with slope −10 (very steep fall). Ask: “Which is steeper? Which has the larger slope VALUE?” The answer: the second line is steeper, but its slope value (−10) is less than the first (1) because it is negative. Slope value and steepness are not the same thing when signs differ.

Beat 8 — Slope as rate of change SHOW: Board: 300 km in 3 hours. Write the slope calculation beside it. SAY: - “Slope is a ratio of two quantities — rise divided by run.” - “In a real-world context, slope becomes a RATE OF CHANGE. The units matter.” - “300 km in 3 hours: slope = 300 km / 3 h = 100 km/h.” - “The units of slope are: units of y per unit of x. Kilometres per hour.” CUE: ★ Write: “m = 300 km / 3 h = [100] km/h. The slope tells you: for each hour, you travel [100] km.”

SHOW: Board: Alberta context — Rocky Mountain highway descends 120 m over 800 m horizontal. SAY: - “Let’s use a real Alberta context. A highway in the Rockies.” - “Rise = −120 m (it goes DOWN, so negative). Run = 800 m.” - “Slope = −120 / 800 = −0.15.” - “The negative sign means: the highway is descending — for each metre you travel horizontally, you drop 0.15 m in elevation.” CUE: ★ Write: “m = [−120] / [800] = [−0.15]. The slope means [the road descends 0.15 m for every 1 m of horizontal distance].”

Section 2 — Retrieval Check Facilitation

Before releasing: SAY: “Close your notes. Everything closed.” CUE: ↩︎ Check: “Notes closed. Retrieval check begins now.”

After students finish (4–5 min): SAY: “Open your notes. Check and correct.” CUE: ★ Write: “Open your notes. Check and correct now.”

Answer guide: 1. m = (−9 − 3) / (4 − (−2)) = −12 / 6 = −2. 2. Slope = 3. (From the table: y increases by 3 for each increase of 1 in x. Pick any two points: e.g., (0,4) and (1,7) → m = 3/1 = 3.) 3. A horizontal line — completely flat. Context: any quantity that doesn’t change over time: a person sitting still (distance = 0), a room held at a constant temperature, a fixed-rate fee that doesn’t vary. 4. Both are right. 3/−2 = −3/2 = −1.5. The sign and value are identical — just written differently. What matters is that the negative sign is carried through: slope is −1.5. 5. Stretch: slope = (b − 0) / (0 − a) = −b/a. (The line passes through (a, 0) and (0, b). Rise = b − 0 = b, run = 0 − a = −a, so m = b / (−a) = −b/a.)

Section 3 — The Equation y = mx + b

Learning target: I can identify the slope and y-intercept from an equation in slope-intercept form, write an equation given the slope and y-intercept, and interpret both values in context. Timing: ~25 minutes direct instruction

Beat 1 — Vocabulary setup SHOW: Board: The four vocabulary terms — Slope-intercept form / y-intercept / x-intercept / Initial value. SAY: - “Four terms today. We already know ‘slope.’ These build on that.” - “Don’t fill them in yet.” CUE: ⊡ Think: “Which of these four terms sounds most familiar? Keep it in your head.”

Beat 2 — Anatomy of y = mx + b SHOW: Board: Write y = mx + b large. Draw arrows pointing to m and b. Label: m = slope (rate of change) and b = y-intercept (starting value). SAY: - “y = mx + b. This is called slope-intercept form.” - “m is the slope — the rate of change, the steepness, the rise over run. Everything we just learned.” - “b is the y-intercept — the value of y when x equals zero. The starting point.” - “On the graph: b is where the line CROSSES the y-axis.” - “Once you can read m and b out of an equation, you can draw the line immediately.” CUE: ★ Write: “Complete the y = mx + b breakdown table — m = rate of change, b = starting value / where line crosses y-axis.”

FLAG — Misconception“The y-intercept is where the line crosses the x-axis.” This is one of the most persistent errors in linear relations. Address it directly now and again when you reach the misconception block in the student notes. The y-intercept is on the Y-axis — where x = 0. The x-intercept is on the X-axis — where y = 0. Two different axes. Two different intercepts.
FLAG — Misconception“Slope is the y-intercept.” In y = mx + b, students often mix up which letter is which. Establish the convention clearly: m is the multiplier (it multiplies x), b is the bonus (added on). These are memory tricks — the real understanding is what each parameter does on the graph.

Beat 3 — Reading the equation directly SHOW: Board: y = 3x − 4. Draw the worked example box. SAY: - “Read this equation. What is the slope?” - Pause 3 seconds. [m = 3]. - “What is the y-intercept?” - Pause. [b = −4]. - “The starting point is (0, [−4]). For each x that increases by 1, y changes by [3].” - “Notice: the equation tells us everything we need to draw the graph. Start at (0, −4). Move up 3, right 1. Up 3, right 1. Connect the dots.” CUE: ★ Write: “Slope m = [3], y-intercept b = [−4], starting point (0, [−4]), y changes by [3] per unit of x.”

Self-explanation beat: SAY: “Before moving on — in your own words: why is b called the y-intercept and not just ‘the constant’? What does it show on the graph?” CUE: ★ Write: “Answer the explain prompt — why is b called the y-intercept?”

You want: b is the value of y when x = 0. That point (0, b) is exactly where the line crosses the y-axis — the vertical axis. ‘Intercept’ means ‘crosses.’

Beat 4 — Write the equation from context (Alberta setting) SHOW: Board: Banff hotel: $90/night + $25 one-time parking fee. SAY: - “Let’s write the equation. What is the starting value — the cost when nights = 0?” - Pause. [$25]. - “What is the rate of change — cost per night?” - Pause. [$90]. - “Equation: C = 90n + 25. What does the 90 represent? What does the 25 represent?” - “90 is the slope — the rate of change — how much cost increases per night.” - “25 is the y-intercept — the fixed parking fee that exists before any nights are counted.” CUE: ★ Write: “Starting value = [$25], rate of change = [$90], Equation: C = [90] n + [25].”

SHOW: Board: *Second point: (1, ___)*. Graph axes already on student page. SAY: - “Plot the y-intercept: (0, 25). That’s our starting point.” - “Use the slope: go up 90, right 1. Second point is (1, [115]). Draw the line.” CUE: ✎ Draw: “Plot (0, 25) and (1, 115). Draw the line through both points.”

FLAG — Rule of FourThis worked example hits all four representations. Verbal (the hotel problem), equation (C = 90n + 25), table (students can build it from the graph), graph (just drawn). SAY: “What representation are we in right now? The graph. What’s the verbal? Hotel cost. What’s the equation? C = 90n + 25. Same relationship — four forms.”

Beat 5 — Misconception: intercepts SHOW: Board: Write the misconception boxed in red: “The intercept is where the line hits zero — that’s the x-axis.” SAY: - “I want to address this directly — it’s a very common error.” - “‘Intercept’ means ‘crosses.’ The y-intercept is where the line CROSSES the Y-AXIS.” - “The y-axis is the vertical axis — where x = 0.” - “The x-intercept is where the line crosses the X-AXIS — where y = 0.” - “In y = 3x − 4: y-intercept is −4 (at the point (0, −4)). Find the x-intercept by setting y = 0: 0 = 3x − 4, so x = 4/3.” - “They are at completely different places on the graph.” SHOW: Board: Work through y = 2x − 6. Find both intercepts side by side. SAY: - “y-intercept: set x = 0. y = 2(0) − 6 = [−6]. Point: (0, [−6]).” - “x-intercept: set y = 0. 0 = 2x − 6, so 2x = 6, x = [3]. Point: ([3], 0).” CUE: ★ Write: “y-intercept (x=0): y = [−6] → point ([0], [−6]). x-intercept (y=0): x = [3] → point ([3], [0]).”

Beat 6 — Finding b from slope and a point SHOW: Board: m = 2, passes through (3, 7). Find b. SAY: - “We know the slope. We know one point. We need to find b.” - “Strategy: the point is ON the line, so its coordinates satisfy the equation.” - “Substitute: y = mx + b → 7 = 2(3) + b.” - “7 = 6 + b.” - “b = [1].” - “Equation: y = [2x + 1].” CUE: ★ Write: “b = [1], Equation: y = [2x + 1].”

SAY: “Before moving on — look at where we substituted (3, 7) for (x, y). In your own words: why are we allowed to do that? What does it mean for a point to be ON the line?” CUE: ★ Write: “Answer the explain prompt — why can we substitute the point’s coordinates?”

You want: if the point (3, 7) is on the line, then it satisfies the equation — plugging in x = 3 must give y = 7. That’s what ‘being on the line’ means algebraically.

Beat 7 — Rule of Four in action: y = −2x + 6 SHOW: Board: y = −2x + 6. Leave space for all four representations. SAY: - “Complete the Rule of Four for this equation.” - “Verbal first: starting at [6], the value decreases by [2] for each unit increase in x.” - “Table: substitute x = 0, 1, 2, 3. y-values are [6, 4, 2, 0].” - “Graph: plot and connect. Does it rise or fall? [Falls — negative slope.]” CUE: ★ Write: “Verbal: starting at [6], decreases by [2] per unit. Table: y = [6, 4, 2, 0].” CUE: ✎ Draw: “Plot and draw the line for y = −2x + 6 on your graph.”

FLAG — Rule of FourAnnounce each representation as you build it. Make the explicit connection: “The −2 in the equation matches the ‘−2 per unit’ in the verbal and the constant difference of −2 in the table and the downward slope on the graph. One relationship, four views.”

Section 3 — Retrieval Check Facilitation

Before releasing: SAY: “Close your notes. Everything closed.” CUE: ↩︎ Check: “Notes closed. Retrieval check begins now.”

After students finish (4–5 min): CUE: ★ Write: “Open your notes. Check and correct now.”

Answer guide: 1. Slope m = −3, y-intercept b = 7. 2. y = 4x − 2. 3. The line passes through (0, 5), so b = 5. Find slope using (0,5) and (2,11): m = (11−5)/(2−0) = 6/2 = 3. Equation: y = 3x + 5. 4. The y-intercept (25) represents the one-time parking fee — the fixed cost that exists even before any nights are booked. It is the initial value when n = 0. 5. Stretch: Both lines have slope 3 — they are parallel (same steepness, same direction). They are different because they have different y-intercepts (+2 vs. −5), so they cross the y-axis at different points and never intersect. Parallel lines have equal slopes.

Section 4 — Direct and Partial Variation

Learning target: I can distinguish between direct and partial variation, recognize each from a table, graph, or equation, and connect them to real-world contexts. Timing: ~25 minutes direct instruction

Beat 1 — Vocabulary setup SHOW: Board: The four terms — Direct variation / Partial variation / Constant of variation / Initial value. SAY: - “Two types of linear relations today. They look similar but behave differently.” - “We’ve already seen examples of both — you might not have noticed which type they were.” CUE: ⊡ Think: “Read the four terms. What do you think ‘direct variation’ means? Keep your guess in your head.”

Beat 2 — Direct variation SHOW: Board: DIRECT VARIATION: y = kx (no constant added). Graph passes through the origin (0, 0). SAY: - “Direct variation: y varies directly with x.” - “The equation has the form y = kx — notice NO b term. No constant added.” - “What happens when x = 0? y = k(0) = 0. Always zero.” - “The graph MUST pass through the origin — the point (0, 0).” - “The ratio y/x is constant for all values — that’s the definition of ‘direct.’” - “Example: gas cost = price per litre × litres. When litres = 0, cost = 0. Direct variation.” CUE: ★ Write: “Direct variation equation: y = [kx] (no constant). Graph passes through [the origin]. Ratio y/x is [constant].”

FLAG — Rule of FourSAY: “We’re introducing both forms today. In the equation: look for whether b exists and is non-zero. In the table: look at the y-value when x = 0. In the graph: look for whether the line passes through the origin or the y-axis at some other point.”

Beat 3 — Partial variation SHOW: Board: PARTIAL VARIATION: y = mx + b where b ≠ 0. Graph crosses the y-axis at b, NOT the origin. SAY: - “Partial variation: y varies partially with x.” - “The equation has a constant b — and b is NOT zero.” - “When x = 0, y = b. The starting value is not zero.” - “The graph crosses the y-axis at b — NOT the origin.” - “Example: phone plan with a monthly fee. When texts = 0, you still pay the fee. That fee is b.” - “The DIFFERENCES in y are still constant — it’s still linear. But the y-axis crossing is not at zero.” CUE: ★ Write: “Partial variation equation: y = mx + b where b ≠ [0]. Graph crosses y-axis at [b, not the origin].”

Beat 4 — Side-by-side comparison SHOW: Board: The comparison table — Feature / Direct variation / Partial variation. Fill in live. SAY: - “Let’s put them side by side. Equation: direct = y = kx, partial = y = kx + b where b ≠ 0.” - “Graph passes through: direct = [origin], partial = [y-axis at b].” - “Example: cost = price × quantity (direct) vs. cost = monthly fee + price × quantity (partial).” CUE: ★ Write: “Complete the comparison table — graph passes through: [origin] / [(0, b)].”

Beat 5 — Identify from a table SHOW: Board: Table A (y-values: 0, 5, 10, 15 for x = 0, 1, 2, 3). SAY: - “Look at Table A. What is y when x = 0?” - Pause. [0]. - “When x is zero and y is zero — the relationship starts at the origin.” - “This is direct variation. Equation: y = [5x].” SHOW: Board: Table B (y-values: 8, 12, 16, 20 for x = 0, 1, 2, 3). SAY: - “Table B. What is y when x = 0?” - Pause. [8]. - “y = 8 when x = 0. Not zero. The line does not pass through the origin.” - “This is partial variation. Equation: y = [4x + 8].” - “How to tell — always look at what happens when x = 0: y = 0 means direct; y ≠ 0 means partial.” CUE: ★ Write: “Table A: [direct], equation [y = 5x]. Table B: [partial], equation [y = 4x + 8]. Rule: look at y when x = 0 — if [y = 0] it’s direct; if [y ≠ 0] it’s partial.”

FLAG — Misconception“If both x and y increase, the relation is direct variation.” This is wrong. Partial variation also has both variables increasing. Direct variation specifically requires the graph to pass through the origin — when x = 0, y must also equal 0. The phone plan with a monthly fee sees costs rising with usage, but it is NOT direct variation because there is a cost even at zero usage. Demonstrate with Table B: x increases, y increases, but it is partial variation.

Beat 6 — Alberta context: two phone plans SHOW: Board: Plan A: $0.15 per text, no monthly fee. Plan B: $10/month + $0.05 per text. SAY: - “Plan A: no monthly fee. When texts = 0, cost = 0. What type of variation?” - Pause. [Direct.] - “Equation: C = [0.15t].” - “Plan B: $10 fee exists even before one text. When texts = 0, cost = $10. What type?” - Pause. [Partial.] - “Equation: C = [0.05t + 10].” SHOW: Board: Calculate the cost for 50 texts and 200 texts for both plans. SAY: - “50 texts: Plan A = 0.15(50) = [$7.50]. Plan B = 0.05(50) + 10 = [$12.50].” - “200 texts: Plan A = 0.15(200) = [$30]. Plan B = 0.05(200) + 10 = [$20].” - “At 50 texts — Plan A is cheaper. At 200 texts — Plan B is cheaper.” - “When do they cost the same? Set them equal and solve.” SHOW: Board: 0.15t = 0.05t + 10 → solve step by step. SAY: - “0.15t = 0.05t + 10. Subtract 0.05t: 0.10t = 10. Divide: t = 100 texts.” - “At exactly 100 texts, the plans cost the same.” CUE: ★ Write: “Plan A: C = [0.15t] (direct). Plan B: C = [0.05t + 10] (partial). Equal at [100] texts.”

Self-explanation beat: SAY: “Before moving on — look at how we set the two equations equal. In your own words: what did we do there and why does that strategy work?” CUE: ★ Write: “Answer the explain prompt — what did we do when we set the equations equal and why?”

You want: setting the two cost equations equal to find where they are the same — the x-value (number of texts) where both plans give the same output. If two expressions are equal, they represent the same value. Solving for t finds the input that makes that true.

Section 4 — Retrieval Check Facilitation

Before releasing: SAY: “Close your notes. Everything closed.” CUE: ↩︎ Check: “Notes closed. Retrieval check begins now.”

After students finish (4–5 min): CUE: ★ Write: “Open your notes. Check and correct now.”

Answer guide: 1. y = 4x is direct variation (no constant term; passes through origin). y = 4x + 3 is partial variation (b = 3 ≠ 0; crosses y-axis at 3). 2. Equation: C = 1.80d + 2.50. This is partial variation — there is a fixed base fare of $2.50 regardless of distance. When d = 0, C = 2.50 (not zero). 3. x=0 gives y=6, so the starting value is 6 (not zero) → partial variation. Rate of change: +4 each two units of x → slope = 2. Equation: y = 2x + 6. 4. Direct variation means y = kx. When x = 0, y = k(0) = 0. The output is always zero at zero input — no fixed starting value exists. This means the relationship begins at the origin by definition. 5. Stretch: both graphs are straight lines with the same slope (same steepness, same direction — parallel lines). The direct variation passes through the origin (0,0). The partial variation crosses the y-axis at some point (0, b) where b ≠ 0. They never intersect (parallel). The lines look identical in slope, but one is shifted up or down from the other by b units.
