Answer Key — Not for Distribution

Section 1

1. √144 = 12; −√49 = −7. √49 is the positive root (+7); −√49 is its negative (−7).

2. Principal square root means the non-negative root only. By convention, the radical symbol √ refers to the positive root. √25 = 5 because we define principal square root as the positive value; ±5 would be the complete solution to x² = 25.

Section 2

1. √49 = 7, √64 = 8, so √50 lies between 7 and 8. 7² = 49, 7.1² = 50.41 → estimate ≈ 7.1.

2. √4 = 2 (exact integer — rational). √5 cannot be expressed as a fraction a/b with integer a, b — its decimal is non-terminating and non-repeating (≈ 2.2360679...), so it is irrational.

Section 3

1. c² = 5² + 7² = 25 + 49 = 74 → c = √74 cm

2. Error: student added the legs before squaring (used a + b instead of a² + b²). Pythagorean theorem states a² + b² = c², not (a + b) = c. Correct: c² = 3² + 4² = 9 + 16 = 25 → c = 5.

Section 4

1. √(9/16) = 3/4; √(4/25) = 2/5 (take square root of numerator and denominator separately)

2. 161/2 = √16 = 4. The exponent 1/2 means "the number that, multiplied by itself, gives 16." Using the same idea: 271/3 = ∛27 = 3 (the cube root of 27).
