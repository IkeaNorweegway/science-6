Answer Key — Not for Distribution

Section 1

1. (−3)⁴ = (−3)(−3)(−3)(−3) = 81;   −3⁴ = −(3⁴) = −81

2. In (−3)⁴, the negative is part of the base — it is multiplied four times (even = positive). In −3⁴, the exponent applies only to 3; the negative is applied after, making it negative.

Section 2

1. All equal 1 (any nonzero base to the power 0 = 1). Note: 0⁰ is undefined.

2. Each step divides by 2: 8 → 4 → 2 → 1. The pattern of dividing by 2 each time the exponent decreases by 1 forces 2⁰ = 1.

Section 3

1. a⁵ × a³ = a⁸;   2³ × 2⁴ = 2⁷ = 128

2. Student confused product of powers (add exponents: x^(2+3) = x⁵) with power of a power (multiply exponents: (x²)³ = x⁶). Product of powers rule: same base, addition. Correct: x² × x³ = x⁵.

Section 4

1. a⁷ ÷ a³ = a⁴;   3⁵ ÷ 3² = 3³ = 27

2. a⁵/a² = (a×a×a×a×a)/(a×a) — cancel two a's top and bottom → a³. The rule works because cancellation removes a factor for each denominator term.

Section 5

1. (x³)⁴ = x¹²;   (2²)³ = 2⁶ = 64

2. (x²)³ = x² × x² × x² = x^(2+2+2) = x⁶. Shortcut: multiply the exponents — (x^m)^n = x^(mn).

Section 6

1. (2x)³ = 2³ × x³ = 8x³;   (3/x)² = 3²/x² = 9/x²

2. (x + 3)² = (x + 3)(x + 3) = x² + 6x + 9, not x² + 9. The exponent distributes over factors (things multiplied together), not terms (things added together). Addition does not follow the same rule as multiplication.

Section 7

1. 2⁻³ = 1/2³ = 1/8;   (1/3)⁻² = 3² = 9

2. Using cancellation: 2³ ÷ 2⁴ = (2×2×2)/(2×2×2×2) = 1/2. Using quotient rule: 2³ ÷ 2⁴ = 2^(3−4) = 2⁻¹. Both methods give the same result → 2⁻¹ = 1/2. Negative exponent means reciprocal.
