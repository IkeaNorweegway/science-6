Surface Area and Volume — Concept Summary

Reference sheet

The Core Distinction

Surface area (SA): Total outer area in cm² — how much material to build the surface.

Volume (V): Space inside in cm³ — how much it can hold.

These are fundamentally different measurements with different units. Do not confuse them.

Formulas

SolidSurface AreaVolume
Rectangular prismSA = 2(lw + lh + wh)V = lwh
CylinderSA = 2πr² + 2πrhV = πr²h
Square-based pyramidSA = s² + 2sl  (l = slant height)V = (1/3)s²h
ConeSA = πr² + πrl  (l = slant height)V = (1/3)πr²h
SphereSA = 4πr²V = (4/3)πr³
Slant height vs. perpendicular height
  • Use perpendicular height (h) for volume
  • Use slant height (l) for surface area of pyramids and cones
  • Connect them: l² = r² + h² (Pythagorean theorem)

Any right prism or cylinder: V = B × h   where B = area of base

Worked Examples

Worked Example 1 — Cylinder   r = 5 cm, h = 8 cm

SA = 2π(5)² + 2π(5)(8) = 50π + 80π = 130π ≈ 408.4 cm²
V = π(5)²(8) = 200π ≈ 628.3 cm³

Why 2πr² in SA? Two circular faces. Why 2πrh? The curved surface unrolls to a rectangle with width h and length = circumference = 2πr.
Worked Example 2 — Cone   r = 3 cm, h = 4 cm. Find slant height first.

l = √(r² + h²) = √(9 + 16) = √25 = 5 cm
SA = π(3)² + π(3)(5) = 9π + 15π = 24π ≈ 75.4 cm²
V = (1/3)π(3)²(4) = 12π ≈ 37.7 cm³
Worked Example 3 — Composite object
Cylinder (r = 4 cm, h = 10 cm) topped by a hemisphere (r = 4 cm).

Volume = cylinder + hemisphere = πr²h + (1/2)(4/3)πr³
= π(16)(10) + (2/3)π(64) = 160π + (128/3)π ≈ 636.7 cm³

For surface area: the flat circle where they join is hidden — subtract it from both.

Common Mistakes

Mistakes → Fixes
MistakeFix
Using slant height in volume formulaVolume uses perpendicular height only

Forgetting to subtract hidden faces in compositesEvery joined face is hidden on BOTH objects
Confusing r and dFormulas use r; if given d, halve it first
