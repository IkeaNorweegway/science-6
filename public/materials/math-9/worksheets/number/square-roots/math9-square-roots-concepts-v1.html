Concept Reference

Key Vocabulary

Key Rules

Principal Square Root

√a is always the non-negative root. √9 = 3, NOT ±3.

Square Roots of Fractions

√(a/b) = √a / √b    → √(9/25) = 3/5

What Distributes

√(a × b) = √a × √b ✓     √(a + b) ≠ √a + √b ✗

Irrational Numbers

√n is irrational unless n is a perfect square.

Pythagorean Theorem

(right triangles only): a2 + b2 = c2    where c = hypotenuse

Estimating √n

Find the two perfect squares on either side of n.
√40: 36 < 40 < 49, so 6 < √40 < 7. Since 40 is closer to 36, √40 ≈ 6.3

Connection to Exponents

√x = x1/2

Worked Example 1 — Evaluating square roots

Evaluate: √(36/49) and √0.81

√(36/49) = √36/√49 = 6/7

√0.81 = √(81/100) = 9/10 = 0.9

Why does √(a/b) = √a/√b? Because (√a/√b)2 = (√a)2/(√b)2 = a/b. It checks out.

Worked Example 2 — Estimating irrational square roots

Estimate √52 to one decimal place.

Perfect squares nearby: 49 < 52 < 64, so 7 < √52 < 8

52 is closer to 49, so √52 is closer to 7.

Try 7.2: 7.22 = 51.84 ≈ 52 ✓    √52 ≈ 7.2

Worked Example 3 — Pythagorean theorem

A right triangle has legs 7 cm and 24 cm. Find the hypotenuse.

a2 + b2 = c2
72 + 242 = c2
49 + 576 = c2
c2 = 625
c = √625 = 25 cm (rational — 625 is a perfect square ✓)

A right triangle has hypotenuse 10 m and one leg 6 m. Find the other leg.

62 + b2 = 102
36 + b2 = 100
b2 = 64 → b = 8 m
