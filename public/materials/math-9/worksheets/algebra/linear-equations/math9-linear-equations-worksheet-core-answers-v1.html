Teacher Answer Key — Not for Distribution

Cover the answer. Try the question. Then check.

1. Solve and verify

(a) x + 13 = 5:

x = −8 | Verify: −8 + 13 = 5 ✓

(b) −4x = 28:

x = −7 | Verify: −4(−7) = 28 ✓

(c) x/5 − 3 = 1:

x/5 = 4 → x = 20 | Verify: 20/5 − 3 = 4 − 3 = 1 ✓

(d) 2x + 9 = −7:

2x = −16 → x = −8 | Verify: 2(−8)+9 = −16+9 = −7 ✓

2. Expand brackets first, then solve

(a) 3(x + 4) = 21:

Distribute: 3x + 12 = 21 → 3x = 9 → x = 3
Verify: 3(3+4) = 3(7) = 21 ✓

(b) −2(x − 5) = 14:

Distribute: −2x + 10 = 14 → −2x = 4 → x = −2
Verify: −2(−2−5) = −2(−7) = 14 ✓

Common Mistake

−2(x−5) ≠ −2x−10. The negative distributes AND flips the sign of −5: −2×(−5) = +10.

(c) 5(2x − 1) = 3(x + 6):

10x − 5 = 3x + 18 → 7x = 23 → x = 23/7
Verify: 5(46/7−1) = 5(39/7) = 195/7; 3(23/7+6) = 3(65/7) = 195/7 ✓

3. Variables on both sides

(a) 4x − 3 = 2x + 9:

2x = 12 → x = 6

(b) 7 − 3x = x + 3:

4 = 4x → x = 1

(c) −(x + 2) = 3x − 10:

−x − 2 = 3x − 10 → 8 = 4x → x = 2
Verify: −(2+2) = −4; 3(2)−10 = −4 ✓

4. Clear fractions first

(a) x/2 + 1/3 = 5/6:

LCD = 6. Multiply: 3x + 2 = 5 → 3x = 3 → x = 1

(b) (3/4)x − 2 = (1/2)x + 1:

LCD = 4. Multiply: 3x − 8 = 2x + 4 → x = 12

(c) 0.3x + 1.5 = 0.9:

×10: 3x + 15 = 9 → 3x = −6 → x = −2

5. Solve inequalities and graph

(a) 2x + 1 > 9:

2x > 8 → x > 4
Number line: open circle at 4, arrow pointing right

(b) −3x ≤ 12:

Divide by −3 — FLIP: x ≥ −4
Number line: closed circle at −4, arrow pointing right

Common Mistake

Forgetting to flip when dividing by −3.

(c) 4 − x < 10:

−x < 6 → flip: x > −6
Number line: open circle at −6, arrow pointing right

6. Identify the error

(a) 3(x + 2) = 15, student writes 3x + 2 = 15:

Error: Only multiplied 3 by x, not by 2 — incomplete distribution.
Correct: 3x + 6 = 15 → 3x = 9 → x = 3

(b) −2x > 8, student writes x > −4:

Error: Divided by −2 but forgot to flip the inequality sign.
Correct: x < −4

7. Write and solve

(a) "Doubled, decreased by 7, equals 19":

Equation: 2x − 7 = 19
2x = 26 → x = 13

(b) Three consecutive integers sum to 54:

Let n = first integer. Equation: n + (n+1) + (n+2) = 54
3n + 3 = 54 → 3n = 51 → n = 17
Integers: 17, 18, 19
Verify: 17+18+19 = 54 ✓

8. Alberta context — Banff hotel

(a) 95n + 35 ≤ 415

(b) 95n ≤ 380 → n ≤ 4

(c) They can stay 4 whole nights. (95×4 + 35 = 415 ≤ 415 ✓)

9. Special cases

(a) 3x + 6 = 3(x + 2):

3x + 6 = 3x + 6 → 0 = 0
Infinitely many solutions — the equation is always true (an identity). Every real number works.

(b) 2x + 5 = 2x − 3:

5 = −3 (the x-terms cancel, leaving a false statement)
No solution — the equation is a contradiction. No value of x can make it true.

10. Solve with correct sign convention

(a) −(x − 4) = 2(x + 1):

−x + 4 = 2x + 2 → 2 = 3x → x = 2/3

(b) (2/3)x − (1/4) = (5/12)x + 1/2:

LCD = 12: 8x − 3 = 5x + 6 → 3x = 9 → x = 3
Verify: (2/3)(3)−1/4 = 2−1/4 = 7/4; (5/12)(3)+1/2 = 5/4+2/4 = 7/4 ✓
