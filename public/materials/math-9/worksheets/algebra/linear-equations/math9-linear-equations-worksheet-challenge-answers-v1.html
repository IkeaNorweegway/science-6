Teacher Answer Key — Not for Distribution

Cover the answer. Try the question. Then check.

Format note: Setup and decision step shown. Final answer listed. Full execution left to student to verify.

1. Error analysis

(a) 4(2x − 3) = 5x + 9, student writes "8x − 3 = 5x + 9":

Error named: Did not multiply −3 by 4 when distributing — applied 4 only to 2x.

Setup: 4(2x−3) = 8x − 12. Then: 8x−12=5x+9 → 3x=21

Final answer: x = 7

(b) −4x ≥ −20, student writes x ≥ 5:

Error named: Forgot to flip the inequality sign when dividing by a negative number (−4).

Setup: Divide by −4 → flip ≥ to ≤

Final answer: x ≤ 5

(c) (x + 3)/2 = 5, student writes x + 3 = 5:

Error named: Multiplied only the numerator (x+3) by 2, not the right side by 2.

Setup: Multiply both sides by 2: x+3 = 10

Final answer: x = 7

2. Translate and solve

(a) "Five more than triple a number is two less than double the number":

Equation: 3x + 5 = 2x − 2
Final answer: x = −7

(b) "When a number is subtracted from its double, the result is the same as half the number increased by 3":

Equation: 2x − x = (1/2)x + 3
→ x = (1/2)x + 3 → (1/2)x = 3
Final answer: x = 6

(c) "Three-quarters of a number, decreased by one-half, equals 6":

Equation: (3/4)x − (1/2)x = 6
LCD=4: 3x − 2x = 24 → x = 24
Final answer: x = 24

3. Full working — name each action

(a) 3(x + 2) − 2(x − 4) = 5(x − 1) + 1

Distribute: 3x+6 − 2x+8 = 5x−5+1 (distribute)

Collect: x+14 = 5x−4 (collect like terms)

Move variables: 18 = 4x (subtract x, add 4)

Isolate: x = 18/4 = 9/2

Verify: LHS = 3(9/2+2)−2(9/2−4) = 3(13/2)−2(1/2) = 39/2−1 = 37/2. RHS = 5(9/2−1)+1 = 35/2+1 = 37/2 ✓

(b) (3/4)x − (1/2)(x − 2) = (5/6)x − 1/3

Setup: LCD=12. Multiply all terms:

9x − 6(x−2) = 10x − 4

9x − 6x + 12 = 10x − 4

3x + 12 = 10x − 4 → 16 = 7x

Final answer: x = 16/7

4. Inequalities — reasoning about the flip

(a) −3x + 6 > −9, student gets x > 5:

Incorrect. Setup: −3x > −15 → divide by −3, flip: x < 5 (arrow left, open circle at 5)

(b) Why the sign flips for negative divisors:

6 > 2. Multiply both by −1: −6 < −2. The inequality reversed. Reason: multiplying by a negative mirrors all numbers across zero — what was on the right is now on the left. The ordering reverses completely.

(c) Does subtracting a negative flip the sign?

No. Subtracting a negative is adding a positive — it shifts all values by the same amount without reversing order. Example: 5 > 3. Subtract −2 (add 2): 7 > 5. Same direction ✓

5. Alberta context — hot chocolate

(a) Inequality:

2.75c > 88

(b) Minimum whole cups:

Setup: c > 88/2.75 = 32. Since c must be a whole number: minimum 33 cups

(c) At $2.25 per cup:

Break-even: 2.25c > 88 → c > 39.1 → minimum 40 cups
Additional cups needed: 40 − 33 = 7 more cups

6. Find the values of x

(a) |x − 3| = 7:

x − 3 = 7 → x = 10 OR x − 3 = −7 → x = −4
Two solutions: x = 10 and x = −4

(b) Rectangle perimeter = 56 cm, length = (3x−2), width = (x+4):

Setup: 2(3x−2) + 2(x+4) = 56 → 6x−4+2x+8 = 56 → 8x+4=56 → 8x=52

x = 6.5

Length: 3(6.5)−2 = 17.5 cm    Width: 6.5+4 = 10.5 cm

Verify: 2(17.5)+2(10.5) = 35+21 = 56 ✓

7. Classify before solving

(a) 2(x + 3) − x = x + 6:

Solve: 2x+6−x = x+6 → x+6 = x+6 → 0=0
Infinitely many solutions ✓ — the equation is an identity (always true).

(b) 3x − 5 = 3(x − 2) + 1:

Solve: 3x−5 = 3x−6+1 → 3x−5 = 3x−5 → 0=0
Infinitely many solutions ✓ — also an identity.

Algebraic signals: One solution → x reduces to a single value. No solution → x cancels, leaving false statement (e.g., 3 = −1). Infinitely many → x cancels, leaving true statement (e.g., 6 = 6).

8. Design a problem — model answer

(Accept any valid Alberta-context problem solved by an equation with rational coefficients that yields x = −3/2.)

Problem: A Banff weather station recorded a temperature of (2/3)x°C at dawn. By noon, the temperature had risen by 4°C and reached 3°C. Find the dawn temperature offset x.

Equation: (2/3)x + 4 = 3

Solution: Multiply by 3: 2x + 12 = 9 → 2x = −3 → x = −3/2

The dawn temperature offset was −3/2°C (one and a half degrees below zero as the reference point).
