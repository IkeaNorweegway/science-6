1. Error analysis

(a) Slope through (2,5) and (6,3): student gets −2

Method is wrong, answer is wrong.

Student put x-values in the numerator and y-values in the denominator (reversed rise and run).

Setup: m = (y₂−y₁)/(x₂−x₁) = (3−5)/(6−2) = −2/4

Final answer: m = −1/2

(b) y = 4x − 6: student says y-intercept is (4, 0)

Error named: Confused the coefficient of x (the slope, 4) with the x-intercept, or confused which axis intercept means which.

Correct: The y-intercept is the value of y when x = 0. Substitute x=0: y = 4(0)−6 = −6.

Correct y-intercept: (0, −6)

(c) y = 3x is partial variation "because coefficient is 3"

Error named: The coefficient 3 is the slope (rate of change), not a constant term. Partial variation requires a non-zero constant b added to the equation.

What makes something partial variation: the presence of a constant term b ≠ 0 in y = mx + b, meaning the graph does NOT pass through the origin. y = 3x passes through (0,0) — it IS direct variation.

2. Two lines, same y-intercept b = 4, slopes 2 and −2

(a) Equations:

y = 2x + 4 and y = −2x + 4

(b) x-intercepts:

y = 2x+4: 0 = 2x+4 → x = −2. x-intercept: (−2, 0)

y = −2x+4: 0 = −2x+4 → x = 2. x-intercept: (2, 0)

(c) Symmetry:

The two lines are reflections of each other across the y-axis — same y-intercept, opposite slopes, x-intercepts equidistant from origin. They form a symmetric "V" shape (or inverted V) with the y-axis as the line of symmetry.

3. Slope 3/4, through (4, 5)

(a) Equation:

Setup: y = (3/4)x + b. Substitute (4,5): 5 = (3/4)(4) + b = 3 + b → b = 2. y = (3/4)x + 2

(b) Does it pass through (0, 0)?

No. The y-intercept is b = 2 ≠ 0. Substituting (0,0): y = (3/4)(0)+2 = 2 ≠ 0.

(c) Does it pass through (8, 8)?

Setup: Substitute x=8: y = (3/4)(8)+2 = 6+2 = 8. Yes ✓ — the point (8, 8) satisfies the equation.

(d) x-intercept:

Setup: Set y=0: 0 = (3/4)x + 2 → (3/4)x = −2 → x = −8/3. x-intercept: (−8/3, 0)

4. Alberta context — two plumbers

(a) Equations and type:

Plumber 1: C = 70h + 50 (partial variation)

Plumber 2: C = 50h + 110 (partial variation)

(b) When do they charge the same?

Setup: 70h + 50 = 50h + 110 → 20h = 60 → h = 3. They charge the same at 3 hours: both charge $260.

(c) 3-hour job:

P1: 70(3)+50 = $260. P2: 50(3)+110 = $260. They cost the same — $260 each. (3 hours is exactly the break-even point.)

(d) Interpretation:

y-intercepts: P1 $50 = call-out fee (cost of showing up before any work); P2 $110 = higher call-out fee.

Slopes: P1 $70/hr = labour rate (expensive per hour); P2 $50/hr = cheaper per hour.

Practical: for jobs shorter than 3 hours, Plumber 1 is cheaper (lower base); for jobs longer than 3 hours, Plumber 2 is cheaper (lower hourly rate).

5. Is the table linear?

x12345
y36111827

Differences in y: 6−3=3, 11−6=5, 18−11=7, 27−18=9

Equal? No → NOT linear.

For a relation to be linear, equal changes in x must produce equal changes in y (constant first differences). Here the differences are 3, 5, 7, 9 — they grow by 2 each time, which is the pattern of a quadratic (second differences are constant), not a linear relation.

6. x-intercept (−3, 0) and y-intercept (0, 6)

(a) Slope:

Setup: m = (y₂−y₁)/(x₂−x₁) = (6−0)/(0−(−3)) = 6/3. m = 2

(b) Equation:

y-intercept = 6, slope = 2. y = 2x + 6

(c) Verify:

(−3, 0): 2(−3)+6 = 0 ✓    (0, 6): 2(0)+6 = 6 ✓

7. Water tank draining

(a) Rate of change and equation:

Setup: Change in V = 850−1200 = −350. Change in t = 10−0 = 10. Rate = −350/10 = −35 L/min.

V = −35t + 1200

(b) When empty?

Setup: 0 = −35t + 1200 → t = 1200/35 = 240/7. t ≈ 34.3 minutes — the tank empties in about 34 minutes and 17 seconds.

(c) Direct or partial? y-intercept meaning?

Partial variation — b = 1200 ≠ 0. y-intercept = 1200 L = the initial volume at time t = 0.

(d) Negative slope:

The negative slope (−35) means the volume is decreasing over time — the tank is losing 35 L every minute.

8. Slope −3/5

(a) Two points using slope definition:

Slope = rise/run = −3/5 means: for every 5 units right, go 3 units down. Starting from any point, e.g., (0, 3): next point (5, 0). Sample points: (0, 3) and (5, 0) — accept any valid pair.

(b) Parallel line through (10, 0):

Setup: Same slope −3/5, substitute (10, 0): 0 = (−3/5)(10) + b = −6 + b → b = 6. y = (−3/5)x + 6

(c) Perpendicular slope:

Perpendicular slopes are negative reciprocals. Negative reciprocal of −3/5 = 5/3. A line perpendicular to this one would have slope 5/3.
