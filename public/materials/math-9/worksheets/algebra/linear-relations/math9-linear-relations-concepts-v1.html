Linear Relations — Concept Summary

Reference Sheet

Key Vocabulary

TermMeaningExample
Linear relationA relationship where equal changes in x produce equal changes in yy = 3x + 2
Slope (m)Rate of change; steepness and direction of a linem = rise/run
y-intercept (b)Where the line crosses the y-axis (x = 0)In y = 2x + 5, b = 5
x-interceptWhere the line crosses the x-axis (y = 0)Set y = 0 and solve
Direct variationLinear relation with y-intercept = 0; passes through originy = kx
Partial variationLinear relation with non-zero y-intercepty = kx + b
Rate of changeHow much y changes per unit increase in x; same as slope$50/hour → m = 50

Key Formulas

Slope from Two Points

m = (y2 − y1) / (x2 − x1)

Slope-Intercept Form

y = mx + b    where m = slope and b = y-intercept

Finding the Equation Given Slope and a Point (x1, y1)

Substitute into y = mx + b and solve for b.

The Rule of Four

Every linear relation can be shown four ways:

FormWhat it looks like
Verbal"Starting at 8, increasing by 3 each time x increases by 1"
Tablex: 0, 1, 2, 3 → y: 8, 11, 14, 17
Equationy = 3x + 8
GraphStraight line, y-intercept 8, slope 3

Worked Example 1 — Find Slope and Equation

Two points: (1, 3) and (4, 12)

Slope: m = (12 − 3)/(4 − 1) = 9/3 = 3

Equation: Use y = mx + b → 3 = 3(1) + b → b = 0

Equation: y = 3x (direct variation — passes through origin)

Worked Example 2 — Partial Variation from Context

A plumber charges $80 call-out + $60/hour.

Starting value (b) = 80 (paid even with 0 hours)

Rate of change (m) = 60 (per hour)

Equation: C = 60h + 80

Type: partial variation (y-intercept ≠ 0)

Common Mistakes

Common Mistakes
MistakeFix
